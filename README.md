# 🌱 AgroConecta

## Conectando o Agro Sustentável à Cidade

### Sobre o Projeto

O AgroConecta é uma plataforma digital educativa desenvolvida para o Concurso Agrinho – Educação Digital.

O projeto tem como objetivo conscientizar os usuários sobre a importância do agro sustentável e fortalecer a compreensão da relação entre o campo e a cidade por meio da tecnologia.

A aplicação foi desenvolvida utilizando HTML, CSS e JavaScript, permitindo que os usuários interajam com conteúdos educativos relacionados à sustentabilidade, preservação ambiental e uso consciente dos recursos naturais.

## Objetivos

- Promover educação digital e ambiental;
- Incentivar práticas sustentáveis no agronegócio;
- Demonstrar a importância da conexão entre campo e cidade;
- Aplicar conhecimentos de programação Front-End;
- Desenvolver uma interface acessível e responsiva.

## Funcionalidades

### 💧 Economia de Água

Simula a quantidade de água que pode ser economizada por meio de práticas sustentáveis.

### 🌿 Quiz Sustentável

Apresenta uma pergunta interativa sobre sustentabilidade no agronegócio.

### 🚜 Agro e Cidade

Exibe informações sobre a importância do agro para o abastecimento das cidades.
